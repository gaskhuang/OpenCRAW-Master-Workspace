重要說明

本次未查得自在生技正式公開年營收 / 營業額數字，因此不以未驗證數字做結論。報告以公開可得資料、產品成熟度、媒體訊號與海外拓銷線索做商業判讀。

一、公司定位判讀

自在生技對外主張「以人為本，永續共好」，更像一個生技/健康品牌平台，而不只是單一產品品牌。這種定位有故事，但若要做海外市場，第一入口不能太散，必須用最能打的主力產品切入。

二、目前最有產品力的方向

核心產品

  • 因特力淨酵素牙膏
  • 成人 / 孕婦 / 兒童 / 寵物分眾產品
  • 乾式刷牙應用

差異化賣點

  • 天然唾液酵素與乳鐵蛋白技術
  • 不只是清潔，而是照護場景方案
  • 可切長照 / 高齡 /吞嚥風險族群

三、海外拓展線索

公開文章顯示，自在生技曾參與 2023 東南亞內需拓銷團，前往印尼、馬來西亞、菲律賓，並吸引 4 家潛力買主，其中已有製藥廠買主討論代理或 OEM 合作可能性。這表示海外並非從零開始，而是已有初步驗證。

四、建議海外市場順序

市場建議原因
馬來西亞華人市場大、健康消費教育成本較低、較適合作為第一站
菲律賓人口基數大，若打入通路，放量機會高
印尼市場最大，但法規、清真、通路與教育成本更高，適合後手放大

五、虧轉盈路線

  1. 主 SKU 聚焦：先聚焦 1~2 個最能打的 SKU，不要全面散打。
  2. 先 B2B 再 B2C：先代理、藥局、照護通路、OEM，先穩定出貨。
  3. 做教育型內容：不要只賣牙膏，要賣「乾式刷牙 / 特殊口腔照護」新解法。
  4. 標準化招商包：英文簡報、型錄、代理條件、試單組合、FAQ。
  5. 從產品公司變方案公司：對外賣的是口腔照護解決方案，不是單純日用品。

六、這次提案要提什麼

  • 海外市場切入順序與打法
  • 主力產品聚焦與 SKU 策略
  • B2B 代理 / OEM 招商提案包
  • 長照口腔照護教育內容矩陣
  • 虧轉盈執行路線圖
