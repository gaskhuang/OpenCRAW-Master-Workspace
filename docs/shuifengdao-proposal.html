品牌結構

  • 前台流量品牌：閱部客
  • 個人信任品牌：水丰刀
  • 變現產品品牌：翻譯泡芙學院

這三層結構是完整的商業漏斗：先用內容吸引、再用人格建立信任、最後用教育產品變現。

公開資料重點

  • 誠品專訪：2020 年已是 42 萬級知識型 YouTuber。
  • 2025 媒體稿：連兩年入選台灣區塊鏈影響力人物。
  • 翻譯泡芙學院官網：閱部客 46 萬+、學員 3000+。
  • 新聞稿口徑：學員已超過 1 萬。
  • 社群公開貼文：至少已開 30–36 屆以上。

SWOT 分析

S 優勢

  • 有大型內容入口，導流成本低。
  • 知識型定位，信任度高於純喊單型 KOL。
  • 區塊鏈教育結合防詐敘事，有市場差異化。
  • 已具課程、社群、內容三位一體基礎。

W 劣勢

  • 品牌仍高度綁定個人。
  • 公開數字口徑不一致，容易引發質疑。
  • 加密貨幣市場波動會反過來影響招生情緒。
  • 若成果交付不夠清楚，退款／評價風險會放大。

O 機會

  • 2025–2027 若市場回溫，教育需求會同步上升。
  • 可切入財商、防詐、投資入門，而非只做幣圈。
  • 可延伸會員制、顧問、企業內訓與研究訂閱。
  • 可從個人品牌升級成講師團 / 團隊制品牌。

T 威脅

  • 政策與監管風險。
  • 市場下跌時課程轉換率下滑。
  • 競品多，且市場充斥更激進的承諾型話術。
  • 品牌信任若受損，影響會直接反映在營收端。

2027 營收預估

以下為情境推估，不是官方數字。依據：公開流量、已知學員規模、開班痕跡、教育產品常見模型。

情境前提預估年營收
保守單次班為主、會員弱、品牌合作普通NT$ 1,200 萬 – 1,800 萬
基準課程常態化 + 會員社群 + 品牌合作穩定NT$ 2,000 萬 – 3,500 萬
樂觀牛市 + 會員制成熟 + 高單價服務放大NT$ 4,000 萬 – 6,000 萬+

最值得看的不是營收數字，而是營收結構

  • 如果 70% 收入都還是單次班，抗風險能力不夠。
  • 如果會員社群和續費收入能拉高，估值會好很多。
  • 如果能增加企業合作與顧問，品牌會從內容 IP 變成商業服務公司。

我會盯的三件事

  1. 會員制有沒有做起來
  2. 單一個人品牌有沒有團隊化
  3. 防詐定位能不能擴到更大財商市場

商業建議

  • 短期：強化課程價值敘事，把「學到什麼」與「避免什麼風險」寫得更清楚。
  • 中期：把翻譯泡芙學院從單次班升級成「主課 + 會員 + 社群 + 陪跑」。
  • 長期：建立第二講師與分析師品牌，降低單人依賴。
一句話商業建議：水丰刀如果只是賣課，會是一個賺錢的個人品牌；如果做出可續費的會員制與團隊化交付，就有機會變成真正可規模化的教育公司。
