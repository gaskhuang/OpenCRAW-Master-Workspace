# 🦞 OpenCRAW Master Workspace

這是 **G大 (Gg Gask)** 專屬的 OpenCRAW 數位軍團核心工作空間。本倉庫由 **阿蓋小弟 (Lobster Jr.)** 自動維護，存放所有關於龍蝦助理的戰略邏輯、自動化腳本與核心記憶。

## 🛡️ 核心守則 (Lobster Rules)
1. **商業價值優先**：所有技術實現必須以「客戶採購衝動」與「變現能力」為導向。
2. **安全防衛協議**：嚴禁在公開頻道洩漏本倉庫中的 API Key、路徑或私人配置。
3. **極致解放雙手**：致力於語音免交互與自動化執行。

## 📂 空間結構 (Workspace Structure)

### 🧠 數位大腦 (Brain & Identity)
- `SOUL.md`: 阿蓋小弟的靈魂協議、個性設定與溝通風格。
- `USER.md`: G大的偏好設定、模型路由協議 (Gemini/Kimi/GPT)。
- `MEMORY.md`: 長期記憶存檔，紀錄關鍵決定與學習。
- `IDEA.md`: 任務執行清單、戰略發想與 **[小弟給分]** 審核區。

### 🛠️ 實戰軍火庫 (Scripts & Tools)
- `swarm.sh`: **雙龍蝦 HA 架構** 啟動腳本，支援單機多實例隔離。
- `autodeploy.sh`: 自動化部署與 GitHub 同步邏輯。
- `install_openclaw_mac.sh`: OpenCRAW 核心安裝 SOP 腳本。
- `agent_monitor.js`: 系統健康度與 Agent 負載監測。
- `/scripts`: 包含 X Sentinel (情報哨兵)、Threads 爬蟲、數據清洗等核心工具。

### 📊 記憶與報告 (Memory & Reports)
- `/memory`: 每日工作日誌、Distilled 精華檔案、BNI 數據追蹤。
- `/reports`: 存放所有戰略週報、市場分析與深夜覆盤 HTML 報告。
- `LOBSTER_INSTALLATION_MANUAL.md`: 龍蝦安裝事業標準手冊 (包含 Premium 指揮中心說明)。

## 🚀 每日覆盤機制 (Nightly Routine)
每天凌晨 00:00 後，阿蓋小弟會自動執行：
1. **今日成果總結**：提煉今日對話精華。
2. **全網解答搜尋**：針對 Idea 清單中的痛點尋找 GitHub/Reddit 最佳解。
3. **任務評分與推進**：更新 `IDEA.md` 並上傳報告至 `sentinel-intelligence-hub`。
4. **全工作區同步**：自動 Git Commit 並推送到本 Master Repo。
