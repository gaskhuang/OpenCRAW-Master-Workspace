# OpenClaw 安裝服務銷售 SOP

## 一句話定位
我不是幫你裝軟體，我是幫你做一套會替你工作的 AI 分身系統。

## 成交流程
1. 先問對方每天最煩的重複工作是什麼
2. 問他現在資訊來自哪裡（Telegram / LINE / Email / FB / 社群）
3. 問他最想省掉哪段流程
4. 直接示範一個小場景
5. 報價時不要只講安裝，要講「省時間 + 成交 + 穩定」

## 需求訪談題目
- 你每天最花時間的是哪三件事？
- 哪些訊息/資訊你最怕漏掉？
- 現在有沒有固定做的報表或摘要？
- 你要的是提醒型、監控型、還是執行型 AI？
- 你希望它接 Telegram、LINE 還是 Google？

## 三種成交話術
### 話術 1：老闆版
你不是缺一個工具，你是缺一個會幫你盯流程、盯訊息、盯報告的數位分身。

### 話術 2：效率版
這套不是讓你多做事，是把你本來每天重複做的那些事，自動化掉。

### 話術 3：商業版
速度就是成交率。你比別人更快回、更快整理、更快跟進，客戶就更容易成交。

## 報價原則
- 基礎安裝賣入門
- 真正利潤在整合、客製 workflow、後續調教
- 不要只賣功能，賣結果

## 交付定義
- 安裝完成
- 帳號登入
- 模型可用
- 至少 1 條 workflow 可跑
- 客戶知道怎麼用
