# OpenClaw 安裝服務報價方案

## 核心主張
你買的不是一個 bot。
你買的是一套可以幫你：
- 自動回報
- 自動整理資訊
- 自動監控社群/新聞/競品
- 自動接工作流
- 自動幫你省時間的 AI 作業系統

---

## 方案 A｜入門安裝版
**NT$ 6,800**

適合：個人工作者、內容創作者、自由接案者

### 交付內容
- OpenClaw 基礎安裝
- Telegram 對接
- 1 個模型配置
- 基本提醒 / 任務助理
- 1 次遠端交付教學

### 不含
- 複雜爬蟲
- 多平台同步
- 客製化排程

---

## 方案 B｜商務自動化版
**NT$ 12,800**

適合：中小企業主、業務、顧問、社群運營者

### 交付內容
- OpenClaw 完整安裝
- Telegram / LINE / Google Workspace 基礎整合
- 2~3 個模型路由
- 每日/每週自動報告
- 監控型任務（新聞 / Reddit / FB / X 擇一組）
- 1 套客製化工作流
- 遠端交付 + 調教

### 核心賣點
- 幫老闆省時間
- 幫團隊少做重複事
- 幫資訊流進到同一個入口

---

## 方案 C｜老闆分身版
**NT$ 25,800 起**

適合：真的要把 OpenClaw 當生意工具的人

### 交付內容
- 商務版全部功能
- 多 Agent 分工（總部 / 二號 / 三號）
- 長任務 / 監控 / 夜班流程
- 報告自動化
- GitHub 同步 / 知識庫 / 第二大腦整理
- 客製化銷售 / 情報 / 行銷工作流
- 安裝後 30 天微調支援

### 核心賣點
- 不只是能用
- 是直接變成你的 AI 作業團隊

---

## 加購項目
- Mac mini 硬體代購 / 配置
- Google Drive / Gmail / Calendar 整合
- 社群監控（FB / Reddit / X / Threads）
- 業務報價 / FAQ / Landing Page 製作
- AI 安裝服務 SOP / 團隊內訓
- 客製化自動排程

---

## 報價原則
- 安裝只是起點
- 真正收費點在：整合、調教、工作流、穩定交付、持續優化
