# OpenClaw 安裝服務商業資產包

目標：直接拿來賣 OpenClaw / OpenCRAW 安裝服務。

## 內容
- `landing.html`：對外銷售頁
- `quote.html`：報價 / 需求蒐集頁
- `pricing.md`：方案與報價結構
- `faq.md`：常見問題
- `sales_playbook.md`：銷售與成交 SOP
- `marketing_content.md`：文章 / 貼文 / 短文案
- `implementation_checklist.md`：接單後交付 checklist

## 核心定位
- 賣的不是「安裝軟體」
- 賣的是「老闆級 AI 分身 / 自動化工作系統 / 解放雙手」
- 交付內容包含：安裝、模型路由、排程、自動化、通訊整合、維運、報告

## 下一步建議
1. 先把 `landing.html` 放上公開網址
2. 用 `quote.html` 當詢價頁
3. 用 `marketing_content.md` 直接發 FB / Threads / LINE 社群
4. 實際接到單後照 `implementation_checklist.md` 交付
