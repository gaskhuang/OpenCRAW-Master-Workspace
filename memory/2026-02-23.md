### 簡介
這是一份 RAG 系統的「16 問題清單」，本質上是一個「語義防火牆」。在任何 RAG 或 agent 系統外面架一圈檢查點，用於逐格對照問題來源。

### 適用情境
- 自己搭 RAG 知識庫
- 公司內部搜尋助手
- 使用 openClaw 或其他 agent framework 串工具
- 文件助理或客服 bot
- 當出現幻覺、答非所問、資訊遺漏時用來排查

2. Chunk 切法
